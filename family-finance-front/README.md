# family-finance-front

Family Finance frontend — React 18 + Vite + TypeScript SPA, Tailwind + daisyUI.
Dev server **`:5178`** (`/api` → `:8098` proxy).

```bash
npm run dev       # Vite :5178 (host:true — LAN/telefon)
npm run build     # tsc -b && vite build  (tip-tekshiruv + bundle)
npm run lint      # ESLint
npm run cap:sync  # build + cap sync android   (APK: npm run apk:debug)
```
